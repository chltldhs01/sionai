바이맘 네이버 SA 주간보고서 생성기 퍼블릭 배포 안내

이제 이 프로젝트는 로컬 폴더(outputs)에 엑셀을 저장하지 않아도 동작합니다.
생성된 XLSX는 서버 메모리에 잠깐 보관되었다가 다운로드되며,
다운로드가 끝나면 바로 제거됩니다.

가장 쉬운 공개 방식: Render

1. 이 프로젝트를 GitHub 저장소에 올립니다.
2. Render에서 New + > Blueprint 또는 Web Service를 선택합니다.
3. 저장소를 연결합니다.
4. render.yaml을 사용하면 대부분 자동으로 설정됩니다.
5. 배포가 끝나면 Render가 공개 URL을 발급합니다.
6. 그 URL만 공유하면 누구나 접속해서 CSV 업로드 후 엑셀 다운로드가 가능합니다.

기본 배포 설정
- Build Command: npm install
- Start Command: npm start
- Health Check Path: /api/health

중요 참고
- 무료 플랜은 첫 접속 시 몇 초 정도 깨우는 시간이 있을 수 있습니다.
- 다운로드 링크는 생성 후 10분 동안 재사용할 수 있습니다.
- 사용자가 다시 생성하면 새 링크가 발급됩니다.
- 업로드한 CSV는 브라우저에서 서버로 전송되어 보고서 생성에만 사용됩니다.

로컬 실행
- start-report-generator.bat 실행
- 또는 터미널에서 npm start 실행

배포 후 확인할 것
- 메인 화면 접속
- CSV 업로드
- 결과 요약 노출
- XLSX 다운로드
- 엑셀에서 정상 열림 확인

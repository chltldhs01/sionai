바이맘 네이버 SA 주간보고서 생성기

1. 실행 파일
- start-report-generator.bat

2. 실행 방법
- start-report-generator.bat 파일을 더블클릭합니다.
- 브라우저를 열고 아래 주소로 접속합니다.
- http://localhost:3030

3. 사용 방법
- 브랜드명 입력
- 보고 시작일 / 종료일 입력
- 네이버 SA CSV 파일 업로드
- 운영 메모 입력
- 주간보고서 생성 버튼 클릭
- 생성 완료 후 엑셀 다운로드

4. 생성되는 시트
- 광고주 요약
- 전체 요약
- 일자별 추이
- 파워링크 상세
- 쇼핑검색 상세
- 키워드 상세
- 운영 메모
- 원본 데이터
